# Subscription Operations & Churn Analytics Platform

## Objective
Build an operations-focused analytics platform to understand customer churn and prioritize high-risk customers in subscription-based businesses.

## Tools & Stack
- Python (pandas, feature engineering)
- Power BI (executive dashboards)
- Business-driven risk scoring

## Project Structure
- `data/raw` – original dataset  
- `data/processed` – clean and enriched tables  
- `src/` – data processing and analysis scripts  
- `powerbi/` – Power BI dashboard  

## Key Insights
- Low product usage is the strongest churn driver.
- High support interaction significantly increases churn risk.
- Inactivity strongly correlates with churn.
- Pricing and plan type are secondary factors.

## Deliverables
- Executive Overview dashboard
- Operational Drivers analysis
- Actionable customer risk list
